## üõ†Ô∏è Setup & Configuration (Expanded)

### Prerequisites

- Node.js 18.0 or later (recommended: LTS)
- npm or yarn
- Supabase account (for full features)
- OpenAI and Gemini API keys (for AI video generation)

### Installation Steps

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd ai-video-platform
   ```
2. **Install dependencies**
   ```bash
   npm install
   ```
3. **Set up environment variables**
   - Copy `.env.example` to `.env.local`:
     ```bash
     cp .env.example .env.local
     ```
   - Add your API keys and Supabase credentials to `.env.local`:
     ```env
     OPENAI_API_KEY=your_openai_key
     GEMINI_API_KEY=your_gemini_key
     NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
     NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
     SUPABASE_SERVICE_ROLE_KEY=your_service_role_key
     ENABLE_REAL_AI=true
     NEXT_PUBLIC_APP_URL=http://localhost:3000
     ```
4. **Run the development server**
   ```bash
   npm run dev
   ```
5. **Access the app**
   - Open [http://localhost:3000](http://localhost:3000) in your browser.

### Advanced Configuration

- **Custom Domains**: Configure `NEXT_PUBLIC_APP_URL` for production deployments.
- **API Rate Limits**: Adjust rate limiting in API route middleware for your needs.
- **Theming**: Edit `tailwind.config.js` for custom colors, fonts, and breakpoints.
- **Testing**: Run `npm test` to execute all unit and integration tests.
- **CI/CD**: Integrate with GitHub Actions or Vercel for automated deployments.

### Troubleshooting

- **Common Issues**
  - _API Key Errors_: Ensure your `.env.local` is set up and the server is restarted after changes.
  - _Supabase Connection_: Double-check your Supabase URL and keys.
  - _Port Conflicts_: If port 3000 is in use, set `PORT=xxxx` in `.env.local`.
  - _Build Failures_: Run `npm install` to ensure all dependencies are present.
- **Debugging Tools**
  - Use `/api/test-env` to verify environment variables.
  - Check browser console and server logs for error messages.
  - Use React DevTools and Redux DevTools for UI debugging.
